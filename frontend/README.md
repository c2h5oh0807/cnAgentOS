# cnAgentOS frontend

Vue 3 + TypeScript frontend for cnAgentOS, generated from the Vite Vue TypeScript scaffold and extended with Vue Router, Pinia, Element Plus, Vitest, ESLint, and Prettier.

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm test:unit
pnpm lint
pnpm build
```

During development, Vite proxies `/api` and `/health` to `http://127.0.0.1:8080`.
